# Training Results Directory

This directory contains evidence of actual reinforcement learning training runs.

## 📁 Expected Files

After running `training_trl_colab.ipynb` on Google Colab:

### 1. `training_plot.png`
Dual visualization showing:
- **Left**: Reward progression over episodes with 5-episode moving average
- **Right**: Reward distribution histogram

### 2. `baseline_comparison.png`
Bar chart comparing:
- Random policy baseline (~0.0 average reward)
- Trained agent performance
- Shows % improvement

### 3. `training_summary.txt`
Text file with metrics:
- Total episodes
- Average reward
- Standard deviation
- Min/max rewards
- Improvement trend (final 5 vs first 5 episodes)

---

## 🚀 How to Generate These Files

### Step 1: Run on Google Colab
```bash
# Open training_trl_colab.ipynb in Colab
# Enable T4 GPU: Runtime → Change runtime type → T4 GPU
# Run all cells: Runtime → Run all
```

### Step 2: Download Results
In Colab, navigate to `results/` folder in file browser and download all files.

### Step 3: Commit to Repo
```bash
cp ~/Downloads/training_plot.png results/
cp ~/Downloads/baseline_comparison.png results/
cp ~/Downloads/training_summary.txt results/
git add results/
git commit -m "Add training evidence"
git push
```
